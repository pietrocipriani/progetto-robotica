Detailed Description

template<AngularSystem angular_system, size_t size>
struct coord::angular_type< angular_system, size >

The representing type for the given coordinate system.

Note
Only 3D Lie spaces or arbitrary sized Euler spaces can be represented, due to lack of an uniformed interface between quaternions and complexes in Eigen.
