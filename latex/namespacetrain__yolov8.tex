\doxysubsection{Detailed Description}
Very simple script used to fine-\/tune a YOLOv8 model on the blocks dataset. 

The ultralytics library is really great, and automatically\+:
\begin{DoxyItemize}
\item chooses hyperparameters
\item performs augmentation
\item stops training when there is no improvement
\item generates a lot of useful visualizations in the {\ttfamily runs/} folder
\item keeps track of the last and the best models 
\end{DoxyItemize}
