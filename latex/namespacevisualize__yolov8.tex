\doxysubsection{Detailed Description}
Simple script to run YOLOv8 detection on the image specified via command line (first argument), allowing to choose confidence threshold (second argument) and Intersection Over Union threshold (third argument, optional). 

Calls the ultralytics library with show=True to automatically show the detected boxes in a cv2 window.
