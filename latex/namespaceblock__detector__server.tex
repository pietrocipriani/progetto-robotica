\doxysubsection{Detailed Description}
Starts a ROS service that takes an {\ttfamily sensor\+\_\+msgs/\+Image} as input, detects boxes in the image, and returns {\ttfamily block\+\_\+detector/\+Detect\+Blocks}, i.\+e. 

the boxes detected by YOLOv8, as output. A service was preferred, instead of a publisher reading directly from the camera and publishing boxes from time to time, for the following reasons\+:
\begin{DoxyItemize}
\item to allow making this component more standalone
\item to allow the input image to be chosen by the caller, e.\+g. by cropping it or passing a custom image
\item to avoid wasting CPU on calculating boxes that would never be used because the subscriber (if any) does not need them
\end{DoxyItemize}

This file also contains the configuration to pass to the YOLOv8 model for detecting boxes, such as CONFIDENCE\+\_\+\+THRESHOLD and INTERSECTION\+\_\+\+OVER\+\_\+\+UNION\+\_\+\+THRESHOLD.

You can visualize results from this service using {\ttfamily \doxylink{inspect__detected__blocks_8py}{inspect\+\_\+detected\+\_\+blocks.\+py}}.
